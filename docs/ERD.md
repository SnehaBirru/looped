# Engineering Requirements Document

## Overview

Looped is a crochet pattern generator that creates custom patterns based on user inputs.

## Features

- Generate patterns for various crochet items (tote bags, hats, scarves, etc.)
- Calculate materials needed (yarn, hook size)
- Generate step-by-step instructions
- Support for different stitch types
- Gauge-based calculations

## Technical Stack

- **Backend**: FastAPI (Python)
- **Frontend**: Next.js + React + TypeScript
- **Architecture**: Monorepo with separate backend/frontend

## Future Enhancements

- CrochetPARADE integration
- Image upload for pattern matching
- Pattern saving and sharing
- User accounts and pattern history
